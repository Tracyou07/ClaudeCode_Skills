# Code Simplifier

自动简化和优化最近修改的代码，提升清晰度和可维护性，不改变功能。

## 调用

```
/simplify
```

## 核心原则

- **保留功能**：只改写法，不改行为
- **遵循项目规范**：按 CLAUDE.md 标准（ES modules、`function` 关键字、显式返回类型等）
- **提升清晰度**：减少嵌套、消除冗余、改善命名、避免嵌套三元
- **保持平衡**：不过度简化，可读性优先于代码行数
- **聚焦范围**：默认只处理当前会话修改过的代码

## 使用模型

Opus（高质量重构）
